# Xavier's Hybrid Developer Curriculum
## Master Index & Overview

**Goal:** Transition from Power Platform developer to Hybrid Developer earning €2,800+ net/month  
**Duration:** 30 weeks (evenings + weekends, ~15-20 hours/week)  
**Stack:** JavaScript → TypeScript → n8n → NestJS → C#/.NET → Power Platform + Windmill  
**Positioning:** The developer who bridges traditional code with Power Platform — rare in the Belgian enterprise market

---

## Your Competitive Advantage

Most developers don't know Power Platform.  
Most Power Platform developers can't write production code.  
**You will do both.** That's the differentiator.

### Phase 1 — JavaScript/TypeScript Foundations (Weeks 1-6)
**Document:** Phase1-JSFoundations.md

Build on your existing JS strength. Refresh ES6+, deepen TypeScript, build REST APIs with Express. This phase moves faster than average because your foundation is already there.

**Projects:** Type-safe utility library | Multi-source API fetcher | Full REST API with Express

---

### Phase 2 — Databases (Weeks 7-8)
**Document:** Phase2-Databases.md

SQL fundamentals, PostgreSQL, database design. Concepts you know, syntax to solidify.

**Projects:** Library management database | Connect Phase 1 API to PostgreSQL | E-commerce schema design

---

### Phase 3 — n8n Automation (Weeks 9-11)
**Document:** Phase3-N8nAutomation.md

Docker as the vehicle, n8n as the tool, real workplace problems as the target. The ServicesLegers 34-column challenge is the running practical example. Produces a working demo for NMBS/SNCB stakeholders.

**Projects:** GitHub→Discord bot | Daily stats dashboard | Morning briefing | ServicesLegers prototype

---

### Phase 4 — NestJS/TypeScript Backend (Weeks 12-17)
**Document:** Phase4-NestJS.md

Enterprise TypeScript backend. NestJS architecture, TypeORM, JWT auth, Docker deployment. Week 17 connects your NestJS API to n8n workflows — the hybrid integration made concrete.

**Projects:** Task Manager API | Blog API | NestJS + n8n integration workflow

---

### Phase 5 — C#/.NET Track (Weeks 18-23)
**Document:** Phase5-DotNet.md

The enterprise language. ASP.NET Core, Entity Framework, Azure deployment, GitHub Actions CI/CD.

**Projects:** C# mini projects | Event Booking API (full .NET, deployed to Azure)

---

### Phase 6 — Power Platform + Windmill (Weeks 24-26)
**Document:** Phase6-PowerPlatformWindmill.md

Your differentiator phase. Custom connectors linking your APIs to Power Platform. PCF components in TypeScript. Windmill as a code-first automation complement to n8n.

**Projects:** Custom connectors for both APIs | PCF data grid component | Windmill deployment pipeline

---

### Phase 7 — Capstone & Job Launch (Weeks 27-30)
**Document:** Phase7-Capstone.md

Full-stack capstone tying every phase together. Parallel job search targeting Belgian enterprise market.

**Project:** Full-stack application: API + Power Platform + n8n automation + Azure deployment

## Ground Rules

**Windows is primary.** All setup instructions are Windows-first. Linux notes included as secondary.

**No Python.** Assessment confirmed JS/TS knowledge is stronger, and PCF uses TypeScript. Python dropped entirely.

**No Mac instructions.** Removed throughout.

**Build first, polish later.** Every phase produces working code.

**Portfolio over perfection.** A deployed imperfect project beats a perfect local one.
